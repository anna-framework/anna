O que é a Anna Framework.

O framework Anna PHP foi desenvolvido para trazer o que há de mais moderno no desenvolvimento web para PHP com simplicidade e facilidade de utilização de seus componentes.

A Anna foi escrita pensando em simplificar ao máximo o desenvolvimento, trazendo um conjunto de ferramentas simples porém poderosas facilmente acessíveis e com baixíssima curva de aprendizado.

Componentes

Assim como os frameworks mais famosos do mercado, a Anna faz uso de inúmeros componentes consolidados para efetuar trabalhos repetitivos e cansativos. As descrições desses componentes assim como o porque que foram selecionados e links para seus manuais estão também disponíveis no menu ao lado.

Maturidade

Anna é um framework novo e precisa passar por um processo de maturação. Está em constante evolução e em beve terá um manual online explicado como utilizar suas ferramentas.

Instalação

Para utilizar a Anna, basta digitar este comando no console: 

          composer create-project anna-framework/installer
Ela vem com uma série de facilitadores que podem ser acessado através da linha de comando, bastando para isso digitar: 
          anna list

Contribuição

Fique a vontade para participar do projeto, caso tenha alguma idéia para componente ou algo que torne a Anna ainda mais simples, basta clonar! Alterações também podem ser realizadas no instalador do framework, acessar em: https://github.com/anna-framework/installer
