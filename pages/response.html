Response

A classe Response é apenas uma interface para utilização da biblioteca symfony/http-foundation. Ela foi escrita apenas com o intuito de poupar códigos.

Response serve para quando você deseja padronizar o retorno de dados para o navegador, alterando os cabeçalhos de maneira fácil com poucas linhas, por exemplo: 

	//controller
	
	...
	return new Response('string de conteúdo', 200, ['content-type' => 'text/html]);

 

A Anna fornece um atalho quando se deseja enviar json para o navegador, bastando para isso retornar seu array ou objeto diretamente 

	//controller
	
	$array = ['frutas' => [
		'maçã',
		'laranja',
		'banana'
	]];
	
	return $array;

 

Nesse caso, a Anna automaticamente recebe o array ou objeto e empacota em json através da classe response
