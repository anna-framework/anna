O que é a Anna.

O framework Anna PHP foi desenvolvido para trazer o que há de mais moderno no desenvolvimento web para PHP com simplicidade e facilidade de utilização de seus componentes.

Funcionamento simples, para reduzir a curva de aprendizado ao extremo, bastanto para isso, alguns minutos de leitura.

A Anna nasceu da filosofia de não ocultar o que está acontecendo em seu sistema, evitando algoritmos mágicos. Para tanto, a documentação foi escrita para explicar o que faz o quê. Caso não se interesse, pode então apenas utilizar, veja como é fácil:

Antes de rodar o Anna framework, você precisa estar atento aos pré-requisitos do sistema, não é algo complexo, basta ter composer e um php 5.4+ rodando em seu sistema na linha de comando.

Se você não tiver familiaridade composer ou até mesmo com a linha de comando do php basta acessar a área de configuração do ambiente e seguir o passo a passo.

Instalação

Uma vez configurado seu ambiente, basta ir até a pasta onde deseja instalar a Anna via console ou terminal e digitar o seguinte comando: 

    composer create-project anna-framework/installer ./

Este comando irá baixar o instalador e rodar o composer para configurar a pasta vendor com os componentes necessários para rodar o framework. Isso pode demorar um pouquinho dependendo da sua conexão. Após isso basta acessar a URL em seu servidor até a pasta /public e você vera a tela de bem vindo.

A partir de então, você pode seguir os tutoriais ao lado, que lhe ensinam a criar controllers, models, registrar rotas, utilizar os repositorios e tudo o que você precisa para trabalhar com tranquilidade e rapidez.
