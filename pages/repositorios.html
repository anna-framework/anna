Criando Repositórios

Repositório é o padrão adotado pela Anna para camada de acesso ao banco de dados. É um padrão que visa organizar os códigos ao mesmo tempo que remove algumas lógicas do controller deixando-o mais limpo.

Anna oferece um repositório simples, porém funcional, pronto para ser extendido e utilizado. Este repositório é genérico e aceita qualquer model. Para torna-lo específico para um determinado model, basta extendê-lo e informar o nome do model pela propriedade Repository::model.

Para criar um repositório basta dar o comando: 

	anna make:repository Teste

 

Este comando cria o respositório TesteRepository na pasta App/Repositories

Você pode também ao criar o repositório já criar também um model com mesmo nome: 

	anna make:repository Teste -m

 

O comando acima irá criar além do repositório TesteRepository na pasta App\Repositories, o TesteModel na pasta App\Models.

A partir daí, você pode utilizar o repositório no seu controller 

namespace App\Controllers;

use Anna\Controller;
use App\Models\TesteModel;
use App\Repositories\TesteRepository;

class TesteController extends Controller
{
	/**
	 * Injetando o repository através do dependence injection
	 * @inject TesteRepository
	 */
	private $repo;
	
	public function index()
	{
		$teste = new TesteModel();
		$this->repo->save($teste);

		$testeFind = $this->repo->search(['id' => 1]);
		return $testeFind;
	}
}
