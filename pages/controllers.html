Criando Controllers

O esqueleto do controlador de página é simples, não foge ao padrão do mercado. 

    class HomeController extends Anna\Controller{
    	public function index(){
            return view('home');
    	}
    }

 

O importante é que seu controlador tenha o sufixo Controller e extenda Anna\Controller. E deve-se ater ao fato de que o roteador não mapeia a url automaticamente para o seu controlador, há sempre de se registrar a rota dele, para isso acesse: Criando Rotas.

Existe o comando anna para criar controladores mais facilmente e ja com um bloco elegante de documentação, por exemplo o comando: 

    php anna make:controller Usuario

 

Irá gerar uma classe controladora assim: 

    namespace App\Controllers;

    use Anna\Controller;

    /**
     * ------------------------------------------------------
     * UsuarioController
     * ------------------------------------------------------
     *
     * Gerado automaticamente pela Anna
     *
     * @author Cristiano Gomes 
     * @since 02/12/2015
     * @package App\Controllers;
     */
    class UsuarioController extends Controller
    {

    	public function index()
        {
    		//do the thing
    	}

    }
