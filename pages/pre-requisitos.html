Pré Requisitos do sistema

Para que a Anna rode linda, leve e solta, seu ambiente precisa estar devidamente configurado, não é difícil olha só:

PHP 5.4+ e linha de comando

O PHP necessário é o 5.4+, ele já está um pouco defasado, por isso acreditamos que não será muito dificil encontrar servidores para ele caso você queira utilizar hospedagens comerciais em seu projeto. Óbviamente, quanto mais recente a versão do PHP utilizada melhor.

Outra configuração que auxiliará muito, porém não é obrigatória, é disponibilizar o PHP na linha de comando, se você está no linux, não há nada o que fazer, já estará lá. Porém, caso utilize o windows basta ir até a pasta onde está o php.exe, copiar o endereço na barra de endereço do windows, e colocar esse endereço na variavel de ambiente PATH.

No menu inicial do windows, busque por "Variaveis de ambiente" você pode editar tanto as variaveis do sistema quanto do seu usuário, da no mesmo. Procure pela variável PATH, caso não exista, pode criar uma, e adicione o caminho do PHP lá, tomando sempre cuidado de separar esse caminho de outros que possam existir lá com ";".

Para confirmar se seu PHP está rodando na linha de comando, feche os consoles abertos, e abra um novo digite então: 

    php -v

Caso tenha dado algo diferente da imagem a cima, deve-ser verificar as razões do erro.

Composer

Fazer o download manual de todos os pacotes que a Anna utiliza, é dolorido, são vários pacotes cada um com suas próprias dependências. Para evitar essa problemática e manter seus pacotes atualizados, alguém teve a genial idéia de criar o COMPOSER clique aqui para saber mais.

Se você utiliza linux, mais uma vez, basta instala-lo e ele já estará no seu terminal, agora você trabalha com windows, aconselho a utilizar o instalador do composer ao invés de baixar o arquivo composer.phar, pois o mesmo já adiciona o comando composer no seu console no escopo global. Sendo asim, acionando o comando: 

    composer -V

Fazendo os workers trabalharem

Caso você queria utilizar os workers para rodar trabalhos e tempos em tempos, a Anna oferece um mecanismo que visa substituir a tabela de cronjob, lhe dando autonomia para rodar de forma prática seus jobs.

Para tanto, caso você utilize windows ou seu servidor de produção seja windows, é necessário ativar no php.ini a extensão php_dotnet.com.dll, esta biblioteca permite que a Anna gerence seus workers em processos paralelos evitando craches e parada de trabalhos. Esta DLL já vem com o php por padrão, basta ser ativada no php.ini.
