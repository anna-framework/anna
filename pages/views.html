Views

As views são parte importante de todo framework PHP, elas podem ser uma benção ou uma maldição.

Para facilitar a vida dos desenvolvedores já acostumados com as novas engines, a Anna disponibiliza 3 engines modernas disponíveis no mercado e já com grande aceitação, são eles Blade, Mustache e Twig.

Para escolher qual dessas engines deseja utilizar, basta ir em App\Config\view.php e alterar as configurações, e informar na propriedade 'view-engine'. 

/**
 * Para renderizar suas views, você pode utilizar as engines: blade, twig ou mustache
 */
'view-engine' => 'blade',

 

A partir daí, é só criar seus templates de acordo com a engine especificada.

No controller, você irá acessar a view, enviar ou buscar parametros dela da mesma forma independente da engine selecionada: 

//controller

...

public function index()
{
	$setFruta = ['nome' => 'laranja'];
	$this->view->addParam('fruta', $setFruta);

	$getFruta = $this->view->getParam('fruta');

	$setFrutas = ['citrica' => 'laranja', 'passa' => 'uva', 'tropical' => 'banana'];

	//adiciona índice por índice aos parâmetros
	$this->view->addParams('frutas', $setFrutas);

	$tropical = $this->view->getParam('tropical');
}

Você pode configurar a view manualmente para renderizar seu template: 

//controller

public function index()
{
	...
	
	//exibe por exemplo o arquivo App\Views\teste.blade.php
	$this->view->setTemplate('teste');
	
	return $this->view;
}

 

Ou você pode utilizar o helper view() para poupar linhas 

//controller

public function index()
{
	...
	
	return view('teste');
	
	...
